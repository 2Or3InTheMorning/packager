TurboWarp Packager Privacy Policy

Updated July 8th, 2021

The website (packager.turbowarp.org)

We may log anonymous aggregated interaction data such as how many people are using the site at a given time, how many times a button is pressed, how often a feature is used, what operating system and browser is being used, and a user's location to the nearest country. We do not log IP addresses. We do not use persistent identifiers. We are not in the business of serving ads or tracking users. You can opt-out by enabling your browser's "Do Not Track" setting.

The site may contact the Scratch API to download projects. See the Scratch privacy policy for their privacy practices. You can avoid this by only packaging projects from local files.

If you ask the packager to package a project from another remote URL, you are subject to the privacy practices of that website.

The output of the website ("packaged projects")

If a packaged project contains cloud variables and is configured to "Connect to cloud variable server", then by default the packaged project may connect to the TurboWarp cloud variable server. See the primary TurboWarp privacy policy for more information. If you configure the packager to connect to a different server, that is outside our control.

If a packaged project uses Scratch extensions that have Wi-Fi icons (for example, the translate extension), the project may connect to Scratch servers to implement certain blocks. See the Scratch privacy policy for more information.

Manual customizations such as the "Custom JS" feature are outside of our control.

Contact

Any concerns related to privacy or any other matter should be sent to: contact@turbowarp.org
